Hvornår kan Ældresagen ikke bruges?
Ældresagen kan ikke kombineres med andre tilbud, rabatter eller kampagner.
Ingen rabat fredag, lørdag og søndag
Ingen rabat på drikkevarer
Ingen rabat på takeaway

Fødselsdagsburger kan ikke kombineres med Ældresagen
Hvis én gæst ved bordet får gratis fødselsdagsburger, kan resten af bordet ikke bruge Ældresagen.
